    Data Structure

    List

    List is an ordered array. The basic def is list = [a, b, c].\
    Function that is often used:

    • list.append(x), adding x at the tail of the list
    • list.insert(a, x), adding x at position a and moving elements after it back one position.
    • list.pop(x), remove and return the x-th element of the list.
    • list.count(x), return the number of times element x exist in the list.
    • list.sort(reverse = a), sort in ascending order if a is false, else sort in descending order.

    Tuple

    Tuple is an ordered array that can’t be changed. The basic def is tuple = (a, b, c).\
