    Click to see proof 首先$E(\bar{X}_n)=\mu$，且因$X_i$为独立随机变量，故 $$ \text{Var}(\bar{X}_n)= \frac{\sum_{i=1}^{n}\text{Var}X_i}{n^2}=\frac{\sigma^2}{n} $$ 那么利用切比雪夫不等式就有 $$ P(|\bar{X}_n-\mu|>\varepsilon)\leq \frac{\text{Var}(\bar{X}_n)}{\varepsilon^2} = \frac{\sigma^2}{n\varepsilon^2}\rightarrow 0, \ \ \ \ \text{as } n\rightarrow 0 $$
    The Strong Law of Large Number

    当$X_1, X_2, …, X_i,…$ IID，且$E(X_i) = \mu<\infty$，则$\bar{X}_n \xrightarrow{a.s.} \mu$

    大数定理想要刻画的就是在有限期望和方差下，随机变量的均值的分布会向随机变量的期望聚集

    $\chi^2$ Distribution

    Definition: $X_1, \dots, X_n$ are i.i.d and follows $N(0, 1)$. $Y = \sum_{i = 1}^{n}X_i^2 \sim \chi^2(n)$.

    Click to see property 1. EY = n, Var(Y) = 2n. 2. If $Y_1, \dots, Y_n$ are independent and $Y_i \sim \chi^2_{k_i}$, $$Y = \sum_{i = 1}^n Y_i \sim \chi^2({\sum_{i = 1}^n k_i})$$ 3. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), $$Y = \frac{1}{\sigma^2}\sum_{i = 1}^{n}(X_i-\bar{X})^2\sim \chi^2(n-1)$$由于本质上减掉$\bar{X}$会去掉一个自由度，故其生成的$\chi^2$分布也减了一个自由度。 4. As $n \rightarrow \infty$, $\frac{X_n-n}{\sqrt{2n}} \xrightarrow{L} N(0, 1)$.

    t Distribution

    Definition: X and Y are independent and $X \sim N(0, 1), Y \sim \chi^2(n)$. $Z = \frac{X}{\sqrt{\frac{Y}{n}}} \sim t(n)$.

    Click to see property 1. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), $$Z = \frac{\sqrt{n}(\bar{X}-a)}{S} \sim t(n-1)$$ $$S = \sqrt{\frac{1}{n-1}\sum_{i = 1}^n(X_i-\bar{X})^2}$$ 2. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), $Y_1, \dots, Y_m$ are i.i.d and follows N(b, $\sigma^2$), $$Z = \sqrt{\frac{mn(m+n-2)}{m+n}} \frac{\bar{X}-\bar{Y}-(a-b)}{\sqrt{(m-1)S_x^2+(n-1)S_y^2}}\sim t(m+n-2)$$ 3. As $n \rightarrow \infty$, $X_n \xrightarrow{L} N(0, 1)$.

    F Distribution

    Definition: X and Y are independent and $X \sim \chi^2(n), Y \sim \chi^2(m)$. $Z = \frac{X/n}{Y/m} \sim F(n, m)$.

    Click to see property 1. If $X\sim t(n)$, $X^2 \sim F(1, n)$. 2. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), $Y_1, \dots, Y_m$ are i.i.d and follows N(b, $\sigma^2$), $$Z = \frac{S_x^2}{S_y^2}\sim F(n-1, m-1)$$ 3. As $m \rightarrow \infty$ and $X_m \sim F(n,m)$, $X_m \xrightarrow{L} \frac{\chi^2(n)}{n}$.
