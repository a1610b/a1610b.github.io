F Distribution

Definition: X and Y are independent and $X \sim \chi^2(n), Y \sim \chi^2(m)$. $Z = \frac{X/n}{Y/m} \sim F(n, m)$.

Click to see property 1. If $X\sim t(n)$, $X^2 \sim F(1, n)$.

Data Reduction

Data reduction is trying to summarize the whole data set with some key features.

A statistic(统计量) $T(\mathbf{X})$, defines a method of data reduction. So for an experimenter he will get $T(\mathbf{x})$ as a result. In this sense, sample $\mathbf{x} \text{ and } \mathbf{y}$ are the same in this statistic if and only if $T(\mathbf{x}) = T(\mathbf{y})$.

统计量的作用本质上就是将样本映射到$\mathbb{R}$上。

那么问题就在于如何评价一个统计量的优劣？

统计学上有三个常见的对于统计量的要求如下

Sufficiency Principle

A sufficient statistic(充分统计量) for a parameter $\theta$ is a statistic that captures all the information about $\theta$ in the sample. 这是一个较为直观但是无法量化的定义，故引入数学上的定义以量化的来刻画充分统计量。

What is a sufficient statistic?

If $T(\mathbf{X})$ is a sufficient statistic for $\theta$ and $\mathbf{x} \text{ and } \mathbf{y}$ are two sample such that $T(\mathbf{x}) = T(\mathbf{y})$, then the inference on $\theta$ should be the same whether $\mathbf{X} = \mathbf{x}$ or $\mathbf{X} = \mathbf{y}$ is observed.

就是说只要充分统计量相同，那么推测参数必然相同。

一种等价的定义的方式是：

A statistic $T(\mathbf{X})$ is a sufficient statistic for $\theta$ if the conditional distribution of the sample $\mathbf{X}$ given the value of $T(\mathbf{X})$ does not depend on $\theta$.

不妨采用离散的情况来理解，那么条件分布就可以看成对于不同的$\mathbf{x}$的$P_{\theta}(\mathbf{X}=\mathbf{x}|T(\mathbf{X})=T(\mathbf{x}))$ 。那么当$T$是一个充分统计量的时候，一旦知道$T(\mathbf{X})$的值就可以在不知道$\theta$ 的情况下知道$\mathbf{X}$的分布。这里的关键在于$T(\mathbf{X})$是一个随机变量而$T(\mathbf{x})$是一个常数。（troubled me for awhile）

How to verify?

We can start from the last definition based on the conditional distribution to verify.

So we want to verify if $P_{\theta}(\mathbf{X}=\mathbf{x}|T(\mathbf{X})=T(\mathbf{x}))$ is the same for all values of $\theta$. Since $\{\mathbf{X} = \mathbf{x}\}$ is a subset of $\{T(\mathbf{X}) = T(\mathbf{x})\}$, we have:

So we can check if $\frac{p(\mathbf{x} | \theta)}{q(T(\mathbf{x}) | \theta)}$ is a constant as a function of $\theta$.

Then we can easily get theorem:

$p(\mathbf{x}|\theta)$ is the joint pdf or pmf of $\mathbf{X}$ and $q(\mathbf{x}|\theta)$ is the joint pdf or pmf of $T(\mathbf{X})$.

$\frac{p(\mathbf{x} | \theta)}{q(T(\mathbf{x}) | \theta)}$ is a constant as a function of $\theta$ $\Longrightarrow$ $T(\mathbf{X})$ is a sufficient statistics for $\theta$.

We also have the famous factorization theorem to do the same work:

$f(\mathbf{x}|\theta)$ is the joint pdf or pmf of $\mathbf{X}$.

$T(\mathbf{X})$ is a sufficient statistics for $\theta$ $\Longleftrightarrow$ $\exists g(t|\theta) \text{ and } h(\mathbf{x}), \text{ s.t. } \forall \mathbf{x} \and \theta\\f(\mathbf{x}|\theta)=g(T(\mathbf{x})|\theta)h(\mathbf{x})$

Click to see proof 只证明离散情况下的情况。 假设$T(\mathbf{X})$是充分统计量。那么令$g(t|\theta)=P_{\theta}(T(\mathbf{X})=t)$和$h(\mathbf{x}) = P(\mathbf{X} = \mathbf{x}|T(\mathbf{X})=T(\mathbf{x}))$。由$T(\mathbf{X})$是充分统计量，故由定义知$h(\mathbf{x})$与$\theta$无关，那么就有$$\begin{aligned} f(\mathbf{x} | \theta) &=P_{\theta}(\mathbf{X}=\mathbf{x}) \\ &=P_{\theta}(\mathbf{X}=\mathbf{x} \text { and } T(\mathbf{X})=T(\mathbf{x})) \\ &=P_{\theta}(T(\mathbf{X})=T(\mathbf{x})) P(\mathbf{X}=\mathbf{x} | T(\mathbf{X})=T(\mathbf{x})) \\ &=g(T(\mathbf{x}) | \theta) h(\mathbf{x}) \end{aligned}$$于是$\Longrightarrow$得证。
先假设$f(\mathbf{x}|\theta)$可被分解，记$q(t|\theta)$是$T(\mathbf{X})$的pmf。定义$A_{T(\mathbf{x})}=\{\mathbf{y}:T(\mathbf{y}=T(\mathbf{x})\}$。那么就有：$$\begin{aligned} \frac{f(\mathbf{x} | \theta)}{q(T(\mathbf{x}) | \theta)} &=\frac{g(T(\mathbf{x}) | \theta) h(\mathbf{x})}{q(T(\mathbf{x}) | \theta)} \\ &=\frac{g(T(\mathbf{x}) | \theta) h(\mathbf{x})}{\Sigma_{A_{T(\mathbf{x})}} g(T(\mathbf{y}) | \theta) h(\mathbf{y})} \\ &=\frac{g(T(\mathbf{x}) | \theta) h(\mathbf{x})}{g(T(\mathbf{x}) | \theta) \Sigma_{A_{T(x)}} h(\mathbf{y})} \\ &=\frac{h(\mathbf{x})}{\Sigma_{A_{T(x)}} h(\mathbf{y})} \end{aligned}$$利用上个定理可知$T(\mathbf{X})$是一个充分统计量。故$\Longleftarrow$得证
Since the exponential family of distribution is very commonly seen, so we derive its unique form of factorization theorem: > $X_1, X_2, .., X_n$ is iid observation from $f(\mathbf{x}|\mathbf{\theta})$ that belongs to : > $$ > f(\mathbf{x}|\mathbf{\theta})=h(\mathbf{x})c(\mathbf{\theta})\exp(\sum_{i=1}^{k}w_i(\mathbf{\theta})t_i(x)) > $$ > where $\mathbf{\theta} = (\theta_1,...,\theta_d), d \leq k$. Then > $$ > T(\mathbf{X})=(\sum_{j=1}^nt_1(X_j),...,\sum_{j=1}^nt_k(X_j)) > $$ > is a sufficient statistic for $\theta$.

Minimal Sufficient Statistics

There often exist more than one sufficient statistics for each model, so we need to value on which is the best statistics to do our estimation. So the concept of minimal sufficient statistics is introduced as the smallest statistics that can carry all the information. Then we define it as:

A sufficient statistic $T(\mathbf{X})$ is a minimal sufficient statistics if, for any other sufficient statistic $T’(\mathbf{X})$, $T(\mathbf{x})$ is a function of $T’(\mathbf{x})$.

Hypothesis Testing

In a hypothesis testing, we start with a null hypothesis. We will ask if the data is strong enough to reject the null hypothesis. If not we will keep the null hypothesis, else we reject null hypothesis and choose the alternative hypothesis. As a common habit, people use $H_0$ to note for null hypothesis and $H_1$ to note for alternative hypothesis.
