    布朗运动

    定义

    数学上的定义为当一个随机过程${B_t: t\ge0}$有：

    1. $B_{t+s}-B_t \sim N(0, s\sigma^2)$

    2. 对$0<t_1<t_2<\dots<t_n$，$B_{t_n}-B_{t_{n-1}},\dots, B_{t_2}-B_{t_1}$相互独立

    3. B的轨道连续

    对于1，2有等价的定义如下：

    1. ${B_t: t\ge0}$的任意有限维分布都是正态分布。

    2. $EB_t = 0, Cov(X_t, X_s) = t\land s$

    还有一些常用的性质如下：

    1. $\frac{B_t}{\sqrt{t}}$与$B_1$同分布。

    2. $p_s(x, y) = p_s(y-x) = \frac{1}{\sqrt{2\pi s}}e^{-\frac{(y-x)^2}{2s}}$

    泛函

    插入一些泛函的概念以理解不变原理。
    对任意的$T > 0$, 有[0, T]上的所有连续函数记为C[0, T]。在这里的含义其实为全部长度为T的连续轨道。
    定义这个空间中的距离(最大模距离)为： 泛函：一个以轨道为自变量的函数f, $f: C[0, T] \rightarrow \mathbb{R}$
    有界：对于泛函f, $\exists M > 0, \text{ s.t. } |f(\varphi)| \le M \qquad \forall \varphi \in C[0, T]$
    连续：对于泛函f, $\forall \varphi \in C[0, T] \quad \forall \epsilon > 0 \exists \delta > 0, \quad \text{s.t. } \forall\phi, d(\phi, \varphi)<\delta, |f(\varphi)-f(\phi)| < \epsilon.$

    布朗桥

    定义为$\{X_t: 0\le t\le 1\}, X_t = B_t -tB_1$。也可拓展为$\{\tilde{X_t}:0 \le t \le s\}, \tilde{X_t} = B_t -\frac{t}{s}B_s+a$\
    布朗桥的性质：

    1. $EX_t = 0, EX_tX_s = s-ts \text{ when } s\le t$

    2. 布朗桥本质上就是给定$B_1 = 0$条件下的布朗运动。

    3. 布朗桥显然是马氏过程，但是是非时齐的
