基本概念

从数据进行学习和预测的模型方法或算法。主要分为有监督学习和无监督学习

  • 有监督学习：对特定的目标变量进行预测，并且有目标变量的数据可以利用。在监督学习Supervised leanring,有一个变量可以对照，优化是一般有比较明确的目标。E.g. 回归、变量分类

  • 无监督学习：无特定的目标变量，目的是探索数据内的规律。E.g. 聚类

超参数

机器学习模型一般有两种参数，分别为模型参数与超参数。模型参数可以直接通过数据得到，如回归系数，决策点的划分条件等。而参数数是描述模型的结构，通常需要在估计模型前制定，如决策树中的树深度，叶结点个数等。

交叉验证法

简称CV(cross validation)，能更稳健地训练、评估和选择模型方法。\
具体步骤为：

  1. 将数据分为k等分（通常k取10）

  2. 利用前k-1分训练模型，并用最后一份评估该模型

  3. 重复第二步，每次选取不同的样本为测试集

  4. 平均所有k个模型得到最终的模型

分类回归

基本概念

给定数据如$(x_1,y_1), (x_2, y_2), …,(x_n, y_n)$。进行有监督学习：$\hat{y_i} = f(x_i)$。将样本分为训练集与检验集。

常见评估标准为第一类错误与第二类错误，同统计。

决策树

基本介绍

决策树的目标是构造一个决策树用来分类与回归。决策树本质上是大量嵌套的if else决策。\
方法逻辑：首先用最好的变量构成树的根。然后重复的将训练集继续分割，每次使用一个变量，直到得到满意的叶子节点。

划分

划分有以下几个常用的法则：

  • Information Gain(信息增益)：基于信息熵的准则。
    熵（Entropy）用来度量一个随机变量的不确定性，不确定性越大熵越大。数学定义为：

  • Gini Index, 基尼指数准则

优化与评估

模型的评估有：

  • 准确率：$\frac{1}{N}\sum_{i=1}^{N}I_{(\hat{y_i}=y_i)}$\
    sklearn中有方法accuracy_score来计算

  • 查准率（1-P(第一类错误)）和查全率（1-P(第二类错误)）\
    sklearn中有方法precision_recall_curve()

  • F1 score:取值范围为0到1，越大表示模型越好
