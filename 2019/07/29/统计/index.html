    Common Distributions

    $\chi^2$ Distribution

    Definition: $X_1, \dots, X_n$ are i.i.d and follows N(0, 1). $Y = \sum_{i = 1}^{n}X_i^2 \sim \chi^2(n)$.

    Basic Properties

    1. EY = n, Var(Y) = 2n.

    2. If $Y_1, \dots, Y_n$ are independent and $Y_i \sim \chi^2_{k_i}$,

    3. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), 由于本质上减掉$\bar{X}$会去掉一个自由度，故其生成的$\chi^2$分布也减了一个自由度。

    4. As $n \rightarrow \infty$, $\frac{X_n-n}{\sqrt{2n}} \xrightarrow{L} N(0, 1)$.

    t Distribution

    Definition: X and Y are independent and $X \sim N(0, 1), Y \sim \chi^2(n)$. $Z = \frac{X}{\sqrt{\frac{Y}{n}}} \sim t(n)$.

    Basic Properties

    1. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$),

    2. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), $Y_1, \dots, Y_m$ are i.i.d and follows N(b, $\sigma^2$),

    3. As $n \rightarrow \infty$, $X_n \xrightarrow{L} N(0, 1)$.

    F Distribution

    Definition: X and Y are independent and $X \sim \chi^2(n), Y \sim \chi^2(m)$. $Z = \frac{X/n}{Y/m} \sim F(n, m)$.

    Basic Properties

    1. If $X\sim t(n)$, $X^2 \sim F(1, n)$.

    2. If $X_1, \dots, X_n$ are i.i.d and follows N(a, $\sigma^2$), $Y_1, \dots, Y_m$ are i.i.d and follows N(b, $\sigma^2$),

    3. As $m \rightarrow \infty$ and $X_m \sim F(n,m)$, $X_m \xrightarrow{L} \frac{\chi^2(n)}{n}$.

    Maximum Likelihood Estimation

    最大似然估计法：构造似然函数$L(x_1, \dots, x_n; \theta) = \prod_i f(x_i;\theta)$。令似然函数最大的$\theta$就是$\hat{\theta}$。常见方法是解 j为$\theta$的维数。\
    当使用最大似然估计法时需要注意敏感性，有时小幅的数据波动可能会导致估计的巨大变动。\
    注意到当$\eta = g(\theta)$时，给定对$\theta$的最大似然估计$\hat{\theta}$，则有对$\eta$的最大似然估计$\hat{\eta} = f(\hat{\theta})$。

    Methods for Evaluating Estimators

    Bias and Variance

    Bias of an estimation is defined by: $B = E_{\theta}(\hat{\theta}) -\theta$\
    An estimator is unbiased if B = 0. We can extend our definition of unbiased to an estimation to $g(\theta)$.\
    The variance is $V = Var_{\theta}(\hat{\theta})$.\
    Mean squared error(MSE, 均方误差) is $E_{\theta}(\hat{\theta}-\theta)^2$.\
    $MSE = B^2+V$\

    Best Unbiased Estimator

    Not so important now, we care more about MSE in current research.\
    An estimator W is called a unifrom minimum variance unbiased estimator(一致最小方差无偏估计量, UMVUE) for $g(\theta)$ if:

    • $E_{\theta}(W) = g(\theta)$ for all $\theta$

    • if exist $W’$ such that $E_{\theta}(W’) = g(\theta)$ for all $\theta$, then $Var_{\theta} \le Var_{\theta}(W’)$.

    Consistency

    • $\phi$ is a consistent estimation, if for any $\epsilon$: $\lim_n P(|\phi(X_1, \dots, X_n)-g(\theta)| \ge \epsilon) = 0$

    • $\phi$ is a strong consistent estimate, if $P(\lim_n \phi(X_1, \dots, X_n) = g(\theta)) = 1$

    Decision

    A decision is defined its decision rules. A decision rule is a measurable function(statistic) T: X $\rightarrow$ A. Where A is the action set.

    Loss Function

    Statistical decision theory using a loss function $L(\theta,\hat{\theta})$ to measure how good the estimator is.\
    Common loss functions for one-dimensional parameter:

    • squared error loss: $L(\theta,\hat{\theta}) = (\theta - \hat{\theta})^2$

    • $L_p$ loss: $L(\theta,\hat{\theta}) = |\theta - \hat{\theta}|^p$

    • zero-one loss: $L(\theta,\hat{\theta}) = 0$ if $\theta = \hat{\theta}$ else 1

    • Kullback-Leibler loss: $L(\theta,\hat{\theta}) = \int p(x;\theta)\log \left(\frac{p(x;\theta)}{p(x;\hat{\theta})}\right)dx$\
      Often used in machine learning and deep learning fields.
